Wie sieht TaskFuchs aus?

Genau so, wie du es möchtest.

Deine Daten gehören Dir 100%

TaskFuchs sammelt keine Nutzerdaten, keine Analytics, kein Tracking. Deine Aufgaben bleiben auf deinem Gerät – immer. Du behältst die volle Kontrolle.

Bewusst ohne KI

Wir verzichten bewusst auf KI-Funktionen. Warum? Du bist Herr über deine Zeit – nicht ein Algorithmus. Keine automatischen Vorschläge, keine Ablenkung. Nur du und deine Aufgaben.

Jetzt starten – kostenlos

TaskFuchs starten

TaskFuchs ist eine Progressive Web App (PWA). Das bedeutet: Die App läuft direkt im Browser – ohne Installation aus einem App Store. Du kannst sie aber auch wie eine echte App auf deinem Gerät installieren.

So installierst du TaskFuchs:

Chrome Adresszeile → Installationssymbol → "Installieren"
Safari Teilen-Button → "Zum Home-Bildschirm"
Edge Menü (···) → "App installieren"
Firefox Adresszeile → "Als App installieren"
Die App findest du unter: app.taskfuchs.de
